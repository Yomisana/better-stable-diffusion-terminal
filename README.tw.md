## 安裝 Stable Diffusion

現在僅支援 Windows  
🚧可能未來會支援 macOS, Linux🚧
[點我下載軟體也可以!](https://github.com/Yomisana/stable-diffusion-installer/releases/latest/download/Stable-Diffusion-Installer.exe)

## 特色
- [X]自動更新
- [X]瀏覽器擴充插件自動掛勾下載模型
- [X]可以偵測當前環境是否移動到另外一台電腦上進行額外修改
- [X]檢查路徑是否為純英文路徑
- 全球性語言
  - 中文
  - 英文
- 終端機可以用選單的方式簡單操作
